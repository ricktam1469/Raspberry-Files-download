
Bundle Configuration and Properties
=====================================================================

Use this directory to store and retrieve configuration files.

com.ge.dspmicro.gitrepository.cfg - Configuration properties for the Predix Machine Git Repository service
com.ge.dspmicro.httpclient.cfg - Configuration properties for the Predix Machine HTTP client service.
com.ge.dspmicro.hoover.spillway-0.cfg - Configuration properties for the Predix Machine Hoover Spillway factory service.
com.ge.dspmicro.machineadapter.modbus-0.cfg - Configuration properties for the Predix Machine Modbus Machine Adapter
com.ge.dspmicro.machineadapter.modbus-0.xml - Data Specific Configuration properties for the Predix Machine Modbus Machine Adapter
com.ge.dspmicro.machineadapter.opcua-0.cfg - Configuration properties for the Predix Machine OPC-UA Machine Adapter
com.ge.dspmicro.machineadapter.opcua-0.xml - Data Specific Configuration properties for the Predix Machine OPC-UA Machine Adapter
com.ge.dspmicro.opcua.server-0.cfg - OPC-UA server configuration properties for Predix Machine OPC-UA Server Service
com.ge.dspmicro.opcuariver.send-0.cfg - OPC-UA River service configuration
com.ge.dspmicro.wsclient.cfg - Configuration properties for the Predix Machine web socket client service.
com.ge.dspmicro.wsserver.cfg - Configuration properties for the Predix Machine Mobile Gateway WebSocket Server
com.ge.dspmicro.websocketriver.send-0.cfg - Configuration properties for the Predix Machine Websocket River service
com.ge.dspnet.datariver.receive-0.cfg - Configuration properties for the Predix Machine Data River Receiver factory service
com.ge.dspnet.datariver.receive.filehandler.cfg - Configuration properties for the Predix Machine Data River Receiver file handler service
com.ge.dspnet.datariver.send-0.cfg - Configuration properties for the Predix Machine Data River Send factory service
com.ge.dspnet.umf.endpoint.cfg - Endpoint configuration of an OSGi example publish-osgi
org.ops4j.pax.logging.cfg - Configuration properties for the PAX logging.
com.ge.dspmicro.webconsole.techconsole.cfg - Configuration properties for Predix Field Agent Tech Console.
com.ge.dspnet.httptunnel.client.cfg - Configuration properties for the Predix Machine HttpTunnel Client service
com.ge.dspnet.httptunnel.client.protocol-ssh.cfg - Configuration properties for the protocols tunneled through the Predix Machine HttpTunnel Client service
